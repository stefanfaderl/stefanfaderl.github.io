Customer Management System

❮ ❯

I created a CRM-System where a registered user can save, edit and display entries in the database. MySQL is used for the connection to the database. Every user that is logged in can only edit and delete the entries he has created. The user can see when each of his entries has been created or modified. The other users do not see this information.

Math Quiz

❮ ❯

There are nine different tasks to solve. Every task has random numbers and the tasks become more and more difficult. If the player has solved the task correctly, he moves up one level. If the result is wrong, he stays in the same level. The score is saved in the local storage and the corresponding level is visually highlighted with a colour.

jQuery • CSS3 • HTML5
